Описание

В работе реализовано:

  1. Двусвязный список
    • вставка, удаление, очистка
    • получение элемента по индексу
    • конвертация списка в массив, массив в список
    • сортировка выбором и быстрая (Хоара)
    • итератор прямой и обратный
  2. Режимы работы приложения через аргументы командной строки
    • генерация
    • сортировка и ее параметры:
      • тип сортировки (возрастание/убывание)
      • алгоритм сортировки
    • вывод
    • выбор файлов для ввода и вывода

Использование makefile

Переменные сборки makefile

  1. CFLAGS - флаги компилятора
    #режим отладки:
    make CFLAGS=-g

  2. LOGLVL - уровень журналирования

    make LOGLVL=2

    0 - откл. журналирование
    1 - ошибки
    2 - ошибки + предупреждения
    3 - ошибки + предупреждения + отдладочная информация

  3. СС - компилятор
    Примечание: в приложении используется тип данных __uint128_t, поэтому следует использовать gcc или clang

    make CC=clang

Цели makefile

  1. make - сборка (переменные см. Переменные makefile)
    make
  2. make cleanall - очистить все объектные файлы и исполняемый файл
    make cleanall
